| **Extensão**                     | **Principal Funcionalidade**                                | **Diferencial**                                                                                                |
| -------------------------------- | ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **YouTube to NotebookLM**        | Adiciona um botão direto abaixo dos vídeos do YouTube.      | Permite adicionar vídeos a notebooks novos ou existentes com um único clique.                                  |
| **NotebookLM Web Importer**      | Importa artigos, blogs e notícias para o notebook.          | Ferramenta gratuita para capturar conteúdo textual de qualquer site.                                           |
| **Bookshelf Folder Manager**     | Cria um sistema de pastas para organizar notebooks.         | Resolve o problema da interface bagunçada (dashboard) do NotebookLM.                                           |
| **Cortex (Favorita do Autor)**   | Ferramenta "tudo-em-um" com painel de gerenciamento.        | Sincroniza Google Docs em massa e gera um **feed de podcast real** (RSS) para ouvir no celular/Apple Podcasts. |
| **NotebookLM Ultra Exporter**    | Exporta ativos em múltiplos formatos (Markdown, PDF, Word). | Permite baixar chat history, relatórios e infográficos de forma estruturada.                                   |

| **Extensão**              | **Principal Funcionalidade**                          | **Diferencial**                                                                                  |
| ------------------------- | ----------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| **AI Blaze**              | Copiloto de IA multimodelos para qualquer página web. | Sincronização de prompts em tempo real para equipes e gatilhos de expansão de texto.             |
| **Bookshelfie**           | Organização de cadernos em "estantes" (pastas).       | Foco total em privacidade com armazenamento 100% local (Local Storage).                          |
| **Grabbit**               | Captura de links e conteúdos em páginas complexas.    | Funciona em estruturas de Shadow DOM e possui rastro de links já visitados.                      |
| **Cortex**                | Hub de captura e automação de conhecimento.           | Criação de um feed de podcast (RSS) privado com resumos das suas pesquisas.                      |
| **Mindmap Extractor**     | Exportação de mapas mentais do NotebookLM.            | Converte o visual travado do site em formatos editáveis (OPML, XML) para outros softwares.       |
| **Scite**                 | Análise contextual de citações científicas.           | Classifica citações em "apoio", "contraste" ou "menção", revelando o consenso real de um estudo. |
| **Ultra Exporter**        | Exportação de dados em múltiplos formatos.            | Capacidade única de exportar notas diretamente para slides (PPTX) ou vídeo (MP4).                |
| **Unpaywall**             | Acesso a artigos científicos com paywall.             | Método ético e legal (Green Open Access) que localiza versões gratuitas de autores.              |
| **Web Importer**          | Importação em lote de sites e vídeos.                 | Integração nativa com o ecossistema do YouTube e gestão de áudios gerados por IA.                |
| **WebSync**               | Importação de sites completos (Full Site).            | Consegue importar conteúdos protegidos por login (intranetes e cursos) usando cookies ativos.    |
